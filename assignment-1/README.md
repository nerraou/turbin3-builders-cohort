# 🎯 Assignment 1 – Guessing Game (Rust)

This project is part of my Turbin3 bootcamp journey.
In this assignment, I implemented the classic **guessing game** from the Rust book to practice core Rust concepts.

---

## 📚 Reference

This project follows the official Rust tutorial:  
👉 [The Rust Programming Language – Guessing Game](https://doc.rust-lang.org/book/ch02-00-guessing-game-tutorial.html)

---

## 🎯 Objective

The goal of this assignment was to:

- Understand basic Rust syntax
- Work with user input
- Practice control flow (`match`, loops)
- Learn error handling
- Use external crates

---

## 🛠️ Concepts Covered

- Variables and mutability
- Data types (`u32`, `String`)
- Standard input/output
- Random number generation (`rand` crate)
- Pattern matching with `match`
- Error handling with `Result`
- Looping (`loop`)

---

## 🎮 How It Works

- The program generates a random number between **1 and 100**
- The user is prompted to guess the number
- After each guess, the program responds with:
  - 📉 Too small
  - 📈 Too big
  - ✅ Correct (game ends)

---

## ▶️ Running the Project

1. Clone the repository:

```bash
git clone turbin3-builders-cohort
cd assignment-1
```

2. Run the program:

```bash

```
