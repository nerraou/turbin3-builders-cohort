# Anchor Marketplace Core

A decentralized NFT marketplace built with **Anchor** and **MPL Core** on Solana.

## Features

- Initialize a marketplace with configurable fees
- List MPL Core NFTs for sale
- Buy NFTs using SOL
- Buy NFTs using SPL tokens
- Delist NFTs and return them to the owner
- Reward buyers with marketplace reward tokens

## Tech Stack

- Solana
- Anchor
- MPL Core
- SPL Token Interface
- LiteSVM for testing

## Marketplace Flow

1. Initialize the marketplace
2. List an NFT for sale
3. Purchase with SOL or SPL tokens
4. NFT ownership transfers to the buyer
5. Marketplace fees are sent to the treasury
6. Buyer receives reward tokens
